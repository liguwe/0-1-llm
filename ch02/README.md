# 第2章：文本数据处理

## 章节目标

本章将学习如何处理和准备文本数据，为训练语言模型打下基础。

## 核心概念

1. **文本分词 (Tokenization)**
   - 将文本转换为数字序列
   - 理解不同的分词方法
   - 使用 tiktoken 分词器

2. **数据加载 (Data Loading)**
   - 创建数据加载器
   - 批处理和序列截断
   - 理解输入输出的维度

3. **嵌入层 (Embedding Layer)**
   - 将 token ID 转换为向量
   - 理解位置编码
