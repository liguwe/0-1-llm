# 第6章：文本分类微调

## 章节目标

本章将学习如何将预训练的 GPT 模型微调用于文本分类任务（如情感分析）。

## 核心概念

1. **微调基础**
   - 分类任务的数据准备
   - 修改模型输出层
   - 冻结 vs 解冻参数

2. **训练分类器**
   - 交叉熵损失
   - 评估指标（准确率、F1 等）
   - 处理类别不平衡

3. **实践应用**
   - IMDb 情感分析
   - 垃圾邮件分类
   - 其他分类任务

## 学习笔记

### 关键知识点

- **为什么微调有效？**
  - 预训练学到的是通用语言理解
  - 微调将知识迁移到特定任务
  - 比从头训练快得多

- **分类任务的微调策略**
  - 只训练最后的分类层
  - 微调部分 Transformer 层
  - 全模型微调
