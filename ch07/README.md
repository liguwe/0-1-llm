# 第7章：指令遵循微调

## 章节目标

本章将学习如何通过指令微调让模型学会遵循用户指令，这是构建 ChatGPT 风格模型的关键步骤。

## 核心概念

1. **指令数据集**
   - 指令-响应对的格式
   - 数据收集和清洗
   - 不同的指令格式

2. **指令微调**
   - 格式化输入数据
   - 训练策略
   - 评估指令遵循能力

3. **评估和测试**
   - 定性评估（人工检查）
   - 定量评估（使用 LLM 评分）
   - Ollama 本地评估

## 学习笔记

### 关键知识点

- **指令微调的重要性**
  - 让模型理解用户意图
  - 提升模型的实用性
  - 架起预训练和应用的桥梁

- **好的指令数据集特征**
  - 多样化的任务类型
  - 清晰的指令格式
  - 高质量的响应

- **评估挑战**
  - 自动化评估困难
  - 需要人工或 LLM 评分
  - 多维度考虑（相关性、准确性、安全性等）
